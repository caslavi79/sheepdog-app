01 — Logo Usage
The Sheepdog Mark

The logo is a black sheepdog silhouette — the cornerstone of the brand.

!

Light gray (#A0A0A0) fails WCAG on white backgrounds. Use it only for decorative elements, dividers, or disabled states — never for readable text. Steel (#7A8490) fails for body text on white — only use it on dark backgrounds where it passes. Red on black fails for body text — red is an accent on light backgrounds only.

Primary Rule

The brand is black-dominant. Dark tones anchor every layout. Red is used sparingly — CTAs, overlines, alerts. If in doubt, use less red.

Red Usage Limits

Red (#C23B22) is reserved for: CTA buttons, overline labels, active states, and alert badges. Never for body text, large fills, headings, or on dark backgrounds as text.

05 — Imagery & Photography
Visual Direction

No storefront. Imagery comes from real on-site work. Stock is a last resort.

Events Photography

  • Team at events, branded gear, working
  • Candid — never posed or staged
  • Crowd management, bartending in action
  • Nighttime and low-light is authentic
  • Mood: energetic, professional, present

Staffing Photography

  • Workers on-site, logistics in action
  • Avoid nightlife imagery entirely
  • Hi-vis vests, clipboards, organized teams
  • Corporate, warehouse, outdoor work
  • Mood: reliable, organized, capable

Photo Treatment

  • No heavy filters — keep natural
  • Slight contrast boost + desaturation OK
  • 60–80% black overlay for text-over-image
  • Never sepia, heavy vignette, or fake grain

Icons & Illustrations

  • One icon library: Lucide or Heroicons
  • Outline/stroke, 1.5–2px weight
  • Never mix filled + outlined on same page
  • No illustrations, cartoons, clip art — ever
06 — Motion & Animation
How Things Move

Motion should feel physical and intentional. Elements ease in with weight and settle into place. Nothing bounces.

Standard Ease
ease · 300ms
Buttons, links, color changes
Spring Settle
cubic-bezier(.16,1,.3,1) · 500ms
Page elements, cards, modals
Hover Response
ease · 200ms
Hover me — scale + darken

Animation Rules

  • Page load: stagger with 100–150ms delays
  • Scroll: fade-up, 20px translate, 0.6s
  • Hover: 200ms max, scale or color only
  • No bounce, elastic, or overshoot
  • Respect prefers-reduced-motion

What Never Animates

  • Logo — static always. No spin/pulse/breathe.
  • Body text — no fade, slide, or typewriter
  • Form inputs — no animated placeholders
  • Nav position — instant, color transitions OK
07 — Service Line Differentiation
Events vs. Staffing

Both share the parent system. If you cover the logo and can't tell which it is, you've drifted too far.

Sheepdog Events

  • Color: Black-dominant, accent red. Warmer.
  • Type: Bold uppercase H1s. Short, punchy.
  • Imagery: Nightlife, concerts, weddings, bartending.
  • Layout: Dynamic — larger heroes, bolder breaks.
  • Mood: "We've got this."

Sheepdog Staffing

  • Color: Black + more gray/steel. Cooler.
  • Type: Sentence-case H1s OK. Measured.
  • Imagery: Operations, logistics, field teams.
  • Layout: Structured, more whitespace.
  • Mood: "Professional workforce, fully managed."

09 — Layout & Spacing
The 8px Grid

All spacing uses multiples of 8.

Base Unit

8, 16, 24, 32, 48, 64, 80, 96. Never arbitrary values.

Sections

Desktop: 80–96px vert, 64px sides. Mobile: 48px vert, 24px sides. Max: 1200px.

Cards

Padding: 24–32px. Gap: 24–32px. Radius: 6–8px consistent.

Hierarchy Flow

  • Overline → H1/H2 → Subhead → Body → CTA
  • Repeats at page, section, and card level

Alignment

  • Body: always left-aligned
  • Center: hero headlines, CTAs only
  • Right: numeric table columns only
10 — Brand Checklist
Pre-Publish Gate

Nothing goes live without passing every item.

Must Pass

  • Logo meets min size + clearspace
  • Logo on approved background
  • Heading font is Barlow
  • Body font is Source Sans 3
  • Colors from approved palette only
  • All text meets WCAG AA (4.5:1)
  • No dangerous color pairs used for text
  • Red accent: CTAs + overlines only
  • Correct file format for destination
  • Service line tone matches audience
  • 8px grid spacing throughout
  • One H1 per page
  • Body max-width 600–680px
  • Animations respect reduced-motion
  • File naming follows convention

Immediate Fails

  • Logo stretched, rotated, recolored
  • Non-approved font anywhere
  • Off-palette colors (#000000 vs #0C0C0C)
  • Red for body text or large fills
  • Light gray (#A0A0A0) used as readable text on white
  • Steel (#7A8490) for body text on white
  • Staged/generic stock photography
  • Mixed icon styles on same page
  • Illustrations, clip art, cartoons
  • Center-aligned body paragraphs
  • Events material looking corporate
  • Staffing looking like nightlife
  • JPEG logo on website
  • Body text running full-width
  • Bouncy/elastic animations or animated logo
