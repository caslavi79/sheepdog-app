!

Amber accessibility rule: Amber #C9922E fails WCAG AA on white (2.8:1) and fails on cream (2.6:1). Never use Amber as text on light backgrounds. Use Amber Dark #8B6212 for overline text on light (5.4:1 AA on white, 5.1:1 AA on cream). Amber is safe as text on dark (7.1:1 AAA on black) and as button fill with black or white text on top.

06 — Usage Rules & Architecture
How the System Works

The palette is layered: parent base shared everywhere, sub-brand extensions only on sub-brand materials. The two sub-brands never share accent colors.

Parent / Hub

Used on sheepdogtexas.com hub page, shared materials (about, contact), ops app, internal docs.

  • Neutrals: Midnight → White (full scale)
  • Accent: Signal Red #C23B22
  • No amber, no navy

Sheepdog Events

Used on /events page, Events social, Events proposals, event-day materials.

  • Parent neutrals + Signal Red (errors only)
  • + Amber, Amber Light, Amber Dark, Copper
  • + Cream, Warm Gray
  • Dual-mode: Security (dark) ↔ Bartending (cream)

Sheepdog Staffing

Used on /staffing page, Staffing social, proposals, B2B materials.

  • Parent neutrals + Signal Red (errors only)
  • + Navy, Slate Blue
  • No amber, no cream, no warm tones
  • Clean, cool, corporate feel
✓

Quick test: If you see amber and cream → Events. If you see navy and slate blue → Staffing. If you see only neutrals and red → Hub. If two sub-brand palettes appear on the same page, something is wrong.

!

Never use off-black as pure black. The brand black is Midnight #0C0C0C, not #000000. Pure black is harsher on screens and creates excessive contrast. Midnight is softer while still reading as black. Same applies to backgrounds — use Charcoal #161616 for secondary dark surfaces, not a random dark gray.
