How to use this form

  • Walk the venue with the Client (or venue manager) before setup begins. Note the condition of each area. Both parties sign the Pre-Event section.
  • Walk the venue again after cleanup is complete. Note the condition of each area. Both parties sign the Post-Event section.
  • Photograph any pre-existing damage before setup and any new issues after cleanup.
  • This checklist is referenced by the Event Setup, Takedown & Cleanup Agreement for damage claims.
